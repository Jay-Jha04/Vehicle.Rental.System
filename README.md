# Vehicle.Rental.System

This application serves as a platform for renting vehicles. It operates as an online vehicle rental store, enabling users to browse available options based on store locations. Users can easily search for a vehicle, be it a car or a bike, and proceed to book it for rent. The booking process involves specifying details such as the desired booking duration, pickup and drop-off locations. Subsequently, users are prompted to make a payment, and upon completion, they receive a booking receipt for their records.
